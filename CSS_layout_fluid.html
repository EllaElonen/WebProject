CSS Layouts: Float

This is a simple example of a float layout including a header (what you´re reading now), two columns/boxes and a footer. Try making your window narrower and see what happens with the sidenavigation bar and the main text column. You should see then stacking up on top of each other.

Float layout is popular because of this ability to adapt to both wide and narrow screens. It takes some design skills to do it in a good way, but it can be well worth it, because it saves you from wasting space.
