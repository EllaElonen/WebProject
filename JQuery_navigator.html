Window navigator

Window navigator (can be written just navigator) is an object with properties for retrieving platform and browser information. Click the button to see your browser version.

Window screen

The screen object has properties that can be used to retrieve information about screen and color qualities. Click the button to see the total height of your screen (shown in pixels).

Window location

The location object has properties that can be used to retrieve information about URL, path, host, protocol etc. Click the button to see the full url of this page.

Window screenLeft, screenTop

The properties screenLeft and screenTop returns the x (horizontal) and y (vertical) coordinates of the window relative to the screen.

Window screenX and screenY

This property returns the coordinates of a window, in relation to the screen. Click the button to see an example.

Alert, prompt and confirm

The window object has methods for creating small alert, prompt and confirmation windows. To see examples, click the buttons.
