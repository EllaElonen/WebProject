Interactive Maps

This picture illustrates the map element, and how it can be made interactive. Click on different parts of the picture to make different things happen! This is all done inside the map element of HTML, by attaching references to different objects, and defining an area attached to a particular alternative. For example: <area shape="rect" coords="0,0,100,200" href="images/BlowHeart.jpg" alt=Magnolia">

More on maps

The map tag is a simple way to build interactive pictures and maps. You will find another similar example, and more information here: W3Schools
