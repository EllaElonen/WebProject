CSS transition

You can change an element the user hovers over just with pure CSS. The pseudo class :hover can be used for this. For example, see what happens to the div element below.

The div selector in the CSS file has a width of 50 pixels, and a one second long transition for width. div:hover defines the new width. This way, you can change the element you hover over, but if you want to change another element, you will need Javascript to do the job.
