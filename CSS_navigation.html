Navigation links and how to style them with CSS

Links

Links to other HTML ages are called hyperlinks. They are surrounded by the anchor tag, <a>. They can be placed into text in regular paragraphs, like this: CSS Basics or added to more complex elements like meny bars, like the menu you can open from the top left corner of this page.

The :active selector styles links that have been made active by clicking. A:hover styles links that the mouse pointers hovers over. See what happens with this link when you hover over it, and when you click on it, before the link page is opened. After yuou have visited the link, it changes color to purple. That is done with the a:visited selector. CSS Basics

How to style buttons

Buttons are styled like any other elements. You use a selector to select the button element, and give it properties like background-color, font-size, text-align (normally center), border and border-radius (to make the edges rounded).
