Examples of JavaScript Variables and Objects

Variable is a container that you give a name, and put contents in. You declare it like this: var variableName = value. There are different types of variables for different types of values, for example text and numbers. In JavaScript, you donÂ´t need to explicitly change the variable type, when you change the contents of a variable to another type; JavaScript does that automatically. Press the buttons to see examples of variables and things you can do with them.

Numbers

This example prints a number variable. JavaScript only has one type of number variables, called number. In Javascript, numbers are always double precision floating point numbers. 15 is the max amount of numbers, without a decimal dot, that is still exact.

Dates

This example prints a date variable, that was created by saving a new Date in a variable d. Without any other specification, it gives the date and time right now, in Central European Time.

Arrays

This example prints a variable containing an array object. An array is a collection of something, objects, numbers, text, dates, lists etc. If you press the button on the left, you will be asked for text input, and see your provided input added into an already existing array of strings.

Booleans

This example prints a boolean variable. Boolean gives the truth value of something. You can use it to set the value of an English sentence to true or false, or you can check if a line of JavaScript code has been run or not, if a result is logically correct or not.

Strings

A string is a text, a string of characters. This example prints two string objects attached to each other.

Math objects

This example prints a math object (Math.PI) saved in a variable.

Global objects

JavaScript has three global properties: Infinity, NaN (not a number), and undefined. In this example, we print the result of a calculation that results in infinity (a number divided by zero), find out the type of the result, and if it is a NaN. For comparison, we find out if the text 'string' is a NaN.

RegEx

This example prints a RegEx variable. RegEx stands for a regular expression, and is a sequence of characters used as a search pattern.
