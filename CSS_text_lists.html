How to style your text

You can use the font-family property to specify the font in which you want the element to be presented. There are lots of fonts you can use in CSS, but all are not supported by all browsers. Otherwise, it is possible to choose a preferred font, and a safe option to fall back on. For example, font-family: Arial, Times, sans-serif; will give the browser Arial font-type to try, otherwise it will fall back on Times. The following table presents some examples of fonts. There are three generic families of fonts: Serif, sans-serif, and monospace. Serifs are small end lines in some letters, Monospace means fixed-width letters. that you can choose to have, or go without (sans-serif). Sans-serif might be easier to read on a browser screen.

The font-size property defines the size of the fonts the chosen text will be presented in. The property "font-size-adjust" means that the font will be adjusted, in case the browser needs to go for the second choice of fonts. For example: font-size-adjust: 0.58; (the aspect value of Verdana) is applied in the red text verdana that might be replaced with times.

The font-size can be set to medium, xx-small, medium, small large, xx-large, x-large, smaller (than parent), or larger (than parent). Other options are setting the length to a certain percentage, or relative length, for example 25em. It can be set to be inherited from a parent or set back to initial.

The font style property can be set to italic or oblique. It can also be set to inherit the style from its parent, or that style can be set to initial. For example, you an set the rest of a div element to italic, but a heading to itÂ´s original normal font-style.

The font-variant property specifies if the text should be written in small-caps or normal. Font-weight can be bold, normal or it can be set in numbers in the range 100-900.

The letter spacing property increases and decreases the space between characters. Word spacing does the same with the space between words. The line height property can change the line height (the height of this line is 2).

With the property text-decoration you can write underlined, overlined, line-through and underlined and overlined text.

With text-shadow you can add shadows in different colors to your text.

With text-indent, you can indent text in paragraphs as you like.
