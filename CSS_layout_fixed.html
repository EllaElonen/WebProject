CSS Layouts: Float

This is a simple example of a fixed layout including headers, a footer, a side navigation column and a main column. Try making your window narrower and see what happens with the side navigation bar and the main text column. You should see parts of the main element disappear, and a side scrollbar show up. That illustrates one of the cons of fixed design. On a narrow screen, you may have to scroll back and forth, which makes reading inconvenient. This is part of the reason why a completely fixed design is not very popular.

Fixed design has still some advantages; it guarantees that every user will see things the same way with the same positions and proportions (if they are willing to scroll).
