CSS Layouts: Elastic

This is a simple example of an elastic layout, including headers, two columns and a footer. Try making your window narrower and see what happens with the side navigation bar and the main text column. You might still need a horizontal scrollbar if you make the window very narrow, but you should see the elements and text adapting to some extent.

Elastic layout is popular because of its ability to adapt to the screen size to some extent, without compromizing too much on the desired appearance. It combines some of the best of fixed and fluid design. It takes some design skills to do it well, but it has possibilities for making a user friendly design.
