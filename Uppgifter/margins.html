The Box Model of CSS: Margins, borders,padding and other useful properties.

CSS is based on a box model, illustrated by this paragraph. The element itself, including the empty space around it it called padding, is lightgreen, and around it there is a solid springgreen border. Te margins are the space around the element and its borders. The top and bottom margins are wider, which is why there is more space between this paragraph and the heading, and this and next paragraph. Around the borders of the element there´s a double, seagreen outline. The difference between margins and padding is that margins define the space inside the borders of an element, padding the space outside.

Margins

Margins define the space outside the borders of an element, which means outside the colored area, if you have added background-color to your element. This paragraph has a margin of 70em.

This paragraph defines each margin separately; margin-top: 100px; margin-bottom: 100px; margin-right: 150px; margin-left: 80px;.

Border-radius is the radius of the element's corners. This paragraph has a border-radius of 25px in all corners. Border-radius is defined for each corner clock-wise, starting from top-left.

Padding

Padding is the space around the contents of an element, but still inside its borders (inside the colored area here). For example, the headings on this page have a padding defined, in order to make the colored area around it bigger, so the first letter does not start exactly at the border. The padding of these headings is 20 px on the left (padding-left), and 10 px above and below (padding-top,padding-bottom). This paragraph has an increased padding, compared with other paragraphs (100 px). That shows up as empty space inside the color frame.

This paragraph has padding-left, padding-top, padding-bottom 120 and padding-right 0.

Elements can also have different outlines around them. This paragraph, for example, has a dotted outline, and a solid border to illustrate the difference between border outline. The outline property also has values dashed, solid, double, groove, ridge, inset, outset.

Gradients

Linear gradient means a color change from one color at the top to another color at the bottom, that happens in the direction of a straight line, like in this paragraph. In this paragraph, it changes from magenta to coral, from top to bottom. You could also change the gradient direction, for example:

background-image: linear-gradient(to right, aqua, deeppink) Note: If you want to support some old browsers that don´t support gradient, add background-color before image, for example the color you want to start from.

You can also add three or more colors: background-image: linear-gradient(to bottom right, darkgreen, lightgreen).

This paragraph has the property background-image: linear-gradient(red, blue 10%, green 20%)

This paragraph has the property background-image: linear-gradient(to top left, darkgreen, springgreen, lightgreen).

This paragraph has the property background-image: repeating-linear-gradient(red, magenta 10%, pink 20%);

Radial gradient has a shape instead of a straight line. For a radius gradient you first need to define a size for the the element.(here: height: 150px; width: 200px;) This paragraph has the property background-image: radial-gradient(circle, red, yellow, green);

This paragraph has the property height: 150px; width: 200px;

This paragraph has the property height: background-image: radial-gradient(farthest-side at 60% 55%, lightblue, skyblue, aqua); It also defines the size of the gradient (from the farthest side).
