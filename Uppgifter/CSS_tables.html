Form Layout

Styling forms is basically like styling any other element, with a few specific things you should know. You can use attribute value selectors, like input[type=number], to choose the types of input you want to give a specific style. The same way, you can change other aspects of the input, just like any other element, and give it a font-size and font-style, background-color, padding etc. If you choose the type password for your input,the input is hidden and is only shown by round markers. Some layout properties are specific for forms.

An example of form layout and style

All text input[type=text] in this form has been given the color darkbrown, passwords are lightgreen, numbers red, and the rest is default black. The layout-property box-sizing is set to border-box here, which guarantees that both the padding and borders will fit inside a given size.
