Semantic tags

Semantic tags are used in HTML5, to mark different parts of text so that browsers, search engines etc. can make more sense of them. They are important information for screen readers, but can also be used to add CSS styles. These paragraphs, for example, are written inside a section element surrounding the part abot semantic tags.

Another example of a semantic element is time. Example of the use of time attribute

You can mark words in a paragraph with the <mark> element. In most browsers, the marked

text will by default be presented with a yellow highlighting.
