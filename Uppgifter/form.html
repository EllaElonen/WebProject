Input fields are specified with the <form> tags, containing <input> tags for every input field. The input tag has a type attribute that gives the type of the input, like text or password in the form above. It also has a name attribute that gives the input a name, and an id attribute to be identified by. For longer text input the textarea tag can be used. In the form above the textarea has a placeholder text, specified in the placeholder attribute, giving the user a clue to what kind of input is being collected.

HTML5 has added the some new input types, like email, url, number, range, date, month, week, time, which you see examples of below. datetime-local has been left out here, because many browsers have removed the support for it.
