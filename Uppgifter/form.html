Examples of Forms

In the following, you will see some examples of forms in HTML.

Input types

Input fields are specified with the <form> tags, containing <input> tags for every input field. The input tag has a type attribute that gives the type of the input, like text or password in the form above. It also has a name attribute that gives the input a name, and an id attribute to be identified by. For longer text input the textarea tag can be used. In the form above the textarea has a placeholder text, specified in the placeholder attribute, giving the user a clue to what kind of input is being collected. These input elements do not send any information, but are merely examples of how inputs can look like. For an example of how to send input, see "Sending input" at the end of the page. The first field (first name) gets fokus automatically, because of the autofocus attribute of the first element.
