Semantic Elements for Structure

The following semantic elements are new in HTML5 and are used to give the page a predictable structure, which might be helpful for search engines, screen readers, and even humans reading the HTML code.

Article and section

The article element can be thought of as a newspaper or magazine article, something that can be published independently. The section element is a part of the page that belongs together, but is not necessarily understood independently of the rest of the page. On this page, for example, most of the contents are included in an article tag, apart from the footer and side bar. The artice is divided into sections for each subject covered in the article.

Navigation

The nav tag can be used for surrounding a navigation element, like the navigation bar that you see if you click on "open side menu".

Figure, figcaption

Pictures can be added to a webpage with the img tag, and a source attribute that gives the location of the picture file.

Header and footer

The h3 element right above this paragraph is surrounded by header tags. The purpose or it is to contain headings and other header elements, like banners or pictures accompanying the heading. A footer element is normally found at the end of the page, like the one on this page that contains contact information.
