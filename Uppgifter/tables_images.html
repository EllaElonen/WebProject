Tables, lists and Images

Tables

Tables are written by writing text inside td tags for regular rows, and <th> for headings. Every piece of data inside the <tr> tags is surrounded by a <td>. Background color is added to columns by enclosing them in a <colgroup span="2"> tag. The color is defined in a separate CSS file. Read more about how to style with CSS here Here.
