CSS Layouts: How to make your document suitable for different screens and printing

Adapting a document to different screens

The viewer of your page might be using a desktop, a cellphone, a laptop or a pad for viewing the page, which is important to consider in your CSS design. To be able to adapt the page to different screen sizes, you need to avoid large fixed objects, that would be hard to show on a small screen, and avoid things that depend on a fixed length. You could also apply dfferent styling to differnet screens. You could use the grid view for the adaptation. All HTML elements should have their box-sizing set to border-box. You can also use a viewport meta tag, like this <meta name="viewport" content="width=device-width, initial-scale=1.0">
