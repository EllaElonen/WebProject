The structure of an HTML page

The sturucture of an HTML-page consists of a <!DOCTYPE>-which defines the markup language so that the browser can display content correctly, and >html<-tags containing head and a body. The head contains metainformation about the page, like encoding, author, description, keywords, title, software that was used to generate it, how search engines should treat it, how often the browser should refresh it, etc.

This page contains the following meta tags: author - the person who created it description - a description of the content generator - the software used to generate it keywords - what the search engines should be looking for robots - defining desired behavior for search engines

Regular paragraphs can be written inside the <p> tag.

Here, for example, <meta http-equiv="refresh" content="30"> was used to tell the page to refresh every 30 seconds.
