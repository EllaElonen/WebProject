How to make your HTML look great with CSS

In order to add style to your HTML, you can apply CSS rules to HTML elements. Most of the time it is best to write your CSS rules in a separate document. That way, it is easy to maintain, and you can apply the same style to all your HTML pages if you like. The following link was added to the HTML of this page, at the end of the head element:
<link rel="stylesheet" type="text/css" href="styles/index.css">

Basic CSS syntax

To style your HTML elements, you first choose them with a CSS selector. For example, the following CSS selects all the paragraps in a document, where no other rule is applied.

p{
}

The script above will identify all elements on a certain level, for example all paragraphs. If you just want to choose certain paragraphs, you will have to use a class selector. You could, for example, create a classstyle called .address, and apply it to all paragraphs consisting of an address.

p[attribute]{
}

You can add properties to the chosen elements, like borders. The following will add a bottom border, a line under the level 1 headlines:

h1{
border-bottom: 2px white solid
}
If you only add a border, without specifictions, a frame around the headline will be added.

The basic structure of HTML pages conssists of a root element that is marked by the <html> tag. Inside that, there are two branches, the head and the body. The following CSS chooses the HTML element and adds a background picture to it.

}
