Web fonts

There are different sites offering web fonts for download. This page uses free fonts from Google Fonts. Follow the link below if you want to find out more about their fonts.

The fonts are imported by adding this link to the HTML file:
<link href="https://fonts.googleapis.com/css?family=Montserrat" rel="stylesheet">
Also, the following CSS code has been added in order to use the font Montserrat on this page:

       body{
            font-family: 'Montserrat', sans-serif;
        }
